# Notesss Compose

A simple note app in jetpack compose , where we can add , delete , update and show notes through server (Backend made in django).

### Tech Stack

`Jetpack Compose`
`Retrofit and ktor`
`Dagger Hilt`
`Coroutines and Flow`
`State Flow and Shared Flow`
`Backend in Django`
